## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Node.js 
* npm or yarn 

### Installation & Running

1.  Clone the repo:
    ```sh
    git clone https://github.com/akashjha518/akash-portfolio.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd akash-portfolio
    ```
3.  Install dependencies:
    ```sh
    npm install
    ```
4.  Run the development server:
    ```sh
    npm run dev
    ```
